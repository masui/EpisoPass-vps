EpisoPass らくらく問題作成

昔の知人の記憶を使って簡単にEpisoPass問題を作ることができます。

1. 名前リストを作成

  • 古い友人や知人の名前を30行ぐらい並べて下さい。
  • 適当な名前を混ぜてもかまいません。

2. 質問リストを作成

  • 上の人物のうち10人程度に関する古い記憶を並べて下さい。
  • 他人が知らない秘密の事実で、将来忘れないようなエピソードを書いて下さい。

3. EpisoPass登録

  • ID と シード文字列 を指定して 登録 ボタンを押すとEpisoPassページができます。
    • ID :   シード文字列 :   登録
  • 「Masui_Twitter」というIDの場合https://EpisoPass.com/Masui_TwitterというURLになります。

4. DASパタン登録 (オプション)

  • EpisoPassのDAS(Draw-a-Secret)機能を使うと解答選択が簡単になります。
  • EpisoPassページで正答を選択してから 右上の「DAS」ボタンを押すとDASパタンを登録できます。
  • DASパタンを登録すると https://EpisoPass.com/Masui_Twitter.html のようなURLでDASを利用できるようになります。
